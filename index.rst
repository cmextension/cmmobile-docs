Welcome to CM Mobile's documentation!
===================================================

CM Mobile is a Joomla! component for helping Joomla! and mobile applications communicate with together.
CM Mobile supports Joomla!'s Users component by default, this provides ability to register a Joomla! user account via mobile applications, with CM Mobile's session manager user can login and perform actions in the supported components.
